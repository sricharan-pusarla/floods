📖 How It Works

🏗️

Reservoirs = Processes

Each reservoir acts as a process that needs drainage resources to safely release water.

🚰

Drainage = Resources

Drainage channels (canals, rivers, spillways) are shared resources with limited capacity.

✅

Safe State = No Flood

If a safe sequence exists where all reservoirs can drain, the system is flood-free.

⚠️

Unsafe State = Flood Risk

If no safe sequence exists, releasing water could cause catastrophic flooding.
